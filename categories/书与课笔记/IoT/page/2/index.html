1. 必要知识

1.1 ARM

ARM 3 个含义

  • 公司，不生产芯片
  • 一类处理器
  • 一种处理器架构

ARM 处理器家族

  • 经典，ARM 7、9、11
  • Cortex-M，单片机，偏控制
  • Cortex-A，跑重量级操作系统，高性能，偏处理
  • Cortex-R，实时，汽车制动

ARM 处理器架构，包含指令集与编程模型（基本数据类型、工作模型、寄存器组），同种体系结构有多种处理器

  • V1 - V3 废弃，V4 - V6 经典处理器
  • V7 ，支持 Thumb-2 32 位指令集。ARMv6-M v6 与 v7 结合产生的版本
  • V8 兼容 V7，支持 64 位

指令集

  • CISC，指令长度不定，执行需要多个周期

  • RISC，长度通常固定，大多一个周期。ARM

SOC（片上系统），包含 CPU、存储器、外设

socket

嵌入式应用程序设计综合教程，第 6 章

1. 分层

  • 应用层。提供服务的应用协议、应用程序，跑在用户态。ftp、http、telnet
  • 传输层。应用程序间的通信，决定送给哪个应用程序。tcp、udp
  • 网络层。通过路由实现主机间通信。ip、icmp、igmp
  • 网络接口层。外设的驱动所在的层，用于屏蔽硬件差异。ppp、arp、rarp

对于 7 层模型，应用层 对应 应用层、表示层、会话层。网络接口层 对应 数据链路层、物理层

RFID

asset、asset-net

1. RFID 概述

物联网，感知层、网络层、应用层。RFID 属于感知层

射频识别，Reader 读卡器，Transponder 卡片 免电池、免接触

读卡器发射电磁波传递能量与时序，卡片（RFID 标签）返回信息

频率划分

  • 125K - 134K 低频。穿透能力强，波段没限制，衰减快、距离短，传输慢。畜牧业、门禁
  • 13.56 M 高频。10 cm，支持防冲撞
  • 860M - 915M 超高频。1m
  • 2.4G - 5G 微波

LoRa

asset、asset-net

1. LoRa 概述

优点

  • 远距离
  • 低功耗
  • 低成本
  • 免牌照频段，不给运营商交钱

LoRa 三个参数影响了通信速率、耗电（电池寿命）、通信距离。详细计算可以通过 LoRa Modem Calculator Tool

  • 带宽 BW，越大速率越高，但降低了接收灵敏度
  • 扩频因子 SF，越大信号质量越强，速率下降
  • 编码率 CR，提高数据可靠性

epoll

有时间继续看 https://idea.popcount.org/2017-02-20-epoll-is-fundamentally-broken-12/

以下翻译自 epoll介绍

  • epoll 实例是一个内核数据结构，epfd 与这个实例相关联。
  • 当所有使用 epfd 的进程 close 或 退出后，内核就会销毁 epoll 实例。
  • 通过 epoll_ctl 添加需要监控的 fd，这个 fd 集合叫做 epoll set 或 interest list 。当有 fd 可以IO时，这些 fd 所在的集合叫做 ready list ，显然 ready list 是 interest list 的子集
  • 一个 fd 可以被添加到多个 epoll 实例，close 这个 fd 会将其从所有的 epoll interest list 中移除
  • 对于 select 和 poll 来说需要遍历所有的 fd，对于 epoll 如果 file description ready 就会移到 ready list 中，当调用 epoll_wait 时只需返回 ready list，因此对于 epoll 是 O(number of events that have occurred) ，对于 select 和 poll 是 O(number of descriptors being monitored)
  • file description 和 inode 的概念见下面分割线中内容。epoll interest list 储存的是 file description，因此会出现共享相同 file description 的进程调用 epoll_wait 时也收到通知的情况，这是个不好的设计，所以在 close 前一定要调用 epoll_ctl 来删除 fd 的订阅。详细解释和举例参考 epoll 缺陷

1. 基本概念

蓝牙 4.0 分为两种

  • Basic Rate (BR, often referred to as BR/EDR for Basic Rate/Enhanced Data Rate)
  • Bluetooth low energy (BLE)

同时支持两种的设备为 Bluetooth Smart Ready，比如手机、笔记本。只支持 BLE 的为 Bluetooth Smart，比如使用电池的设备

蓝牙基本概念

BLE 0-39 个信道，每个2Mhz，37、38、39 广播信道，广播信道用来建立连接，其它数据信道进行数据发送
