PWN

CTF

  • 思维能力
  • 快速学习能力
  • 前两者具备了，技术能力是自然而然的

基础

  • 编程语言，c、汇编、脚本语言
  • 数学基础，算法、密码学
  • 脑洞大开
  • 体力和耐力

分类

  • pwn、reverse 对汇编和逆向
  • crypto 对数学和算法
  • web 对技巧和快速搜索能力，发散思维
  • misc 与计算机安全的都有

都要学的内容

  • linux 基础、计算机组成原理、操作系统原理、网络协议分析

  • pwn、reverse、crypto，IDA 工具使用（f5插件），逆向工程、密码学、缓冲区溢出

  • web、misc 网络安全、内网渗透、数据库安全

笔记整理、How To Get Started In CTF、Tools and Resources to Prepare for a Hacker CTF Competition or Challenge、 Google Project Zero 成员教你如何入门搞安全

重点参考：CTFs github 其中的 resource 库

算法

  • 既然要锻炼学习和思维，就应该选择有挑战的方向
  • 算法相比其它可维持长期收益，是共同的基础 我的算法学习之路、程序员该如何学习算法
  • 主线是两本算法书（算法和邓俊辉的书），其次是网上的教程，最后再考虑视频（学堂x 的数据结构和算法训练营）

pwn 入门

软件安全

  • 研究对象：代码（源代码、字节码、汇编）
  • 研究目标：发现漏洞、利用漏洞、修补漏洞、
  • 研究技术：逆向工程、漏洞挖掘与利用、漏洞防御技术

CTF PWN：软件安全的缩影

特点：入门难、进阶难、精通难

工具

  • 静态分析：IDA Pro
  • 动态调试：gdb（with peda or gef），windbg、ollydbg
  • exploit：pwntools、zio

前置技能

  • 汇编语言：程序执行、函数栈帧、函数调用
  • 编译、链接、装载、执行
  • ELF 文件结构
  • Linux 系统相关：文件描述符、系统调用、socket 编程、shell 命令
