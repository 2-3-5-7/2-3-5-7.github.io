    更新一条记录 update stuinfo set age=30, score = 82 where id=1003;
    增加一列 alter table stuinfo add column sex char;
    删除一列 create table stu as select id, name, score from stuinfo;
    drop table stuinfo;
    alter table stu rename to stuinfo;
    数据库设置主键 create table info(id integer primary key autoincrement, name vchar);
  • Primary Key

每个表只有一个主键，两种设置方式

SQL 标准不允许主键为 NULL，但是为了兼容 SQLite 允许 NULL

如果没设置主键 或 主键类型不为integer，那么就会有一个隐含的自增 rowid 列（可以用 WITHOUT ROWID 不使用），如果设置了主键且为 integer，那么主键就作为 rowid，插入记录时没有指定主键，那么新记录的主键值就是表格中主键最大值加 1。达到最大值 9223372036854775807 后就会重用其它可用的值，无可用则报错。
