Root

参考 教你获取完整root权限， 下面用到的文件保存到 环境\手机\mi9se

  1. 下载 开发版，点击多次 MIUI 图标，选择安装包刷入
  2. 下载 BL 解锁工具，申请解锁，要等一周才能解锁
  3. 进入 fastboot 刷入 twrp。没有驱动可以使用 BL 解锁工具 中的 MiUsbDriver
  4. 下载 magisk zip 文件通过 twrp 刷入，也可用 twrp 自带的 root 功能，也是 magisk
  5. 关闭安全中心 -> 病毒扫描的 root 检测 和 支付环境 检测。用 magisk hide 隐藏全球上网等 root 检测应用。关于 magisk hide 的问题
  6. 后续安装完整官方升级前，需确认 recovery 中脚本自动重启未勾选。每次升级后，进入高级选 签名 boot 或 root，防止 recovery 和 root 被恢复
  • 刷开发版和解锁都要清空手机，所以参考下面进行备份
  • root 后备份可以使用钛备份进行彻底备份
  • 长按电源键 + 音量上键进入 recovery，长按电源键 + 音量下键进入 fastboot

备份与还原

  • 备份时，进行本地备份与云备份。本地备份后拷出到 PC，路径见下

  • 还原时，先不登录云账号，进行本地备份的恢复。然后登录云端，进行云备份的还原。还原文件存放位置 内部存储设备\MIUI\backup\AllBackup\日期开头的备份文件夹，没有则新建

小米

  • 备份微信聊天记录，进行桌面和其它内容的云同步
  • 刷入 twrp 见 除Win外软件\手机\mi9se，目前使用的是 twrp_grus.zip ，估计 TWRP-3.3.0-0421-XIAOMI9SE-CN-wzsx150-fastboot.7z 也可以，
  • 三清(Dalvik/Cache/Data)后，刷入 eu 版 (xiaomi.eu_multi_MI9SE_21.2.3_v12-11.zip)，刷入后清除 data 分区，否则进不了系统
  • 联网后用小米账户解锁，然后关路由器、拔 SIM 卡来跳过谷歌激活，参考
  • 在 recovery 格式化内部存储后，拷入 magisk，并刷入
  • 安装 除Win外软件\手机\mi9se 里面的小米应用商店、MX 播放器、系统应用删除
  • 删除系统应用，视频、音乐、搜索、服务与反馈、浏览器
  • 使用云同步恢复应用，Google Play 恢复应用
  • 安装 magisk (Magisk-v21.4.zip) 设置 magisk hide，全球上网（一元流量）、全球上网组件、支付宝等金融 App
  • 字体修改，默认 nato sans 在同一个作者中的找到 milan pro 安装并应用
  • 不要用自带应用商店更新天气，会导致天气打不开
  • 缺陷：没有公交卡、短信通知类不归类、电话没有来电识别，可以通过 Magisk 模块 解决，但该模块目前会导致无法双开应用。目前公交卡通过小米手环实现，暂时不折腾

极米Z4X

花 50 买的这个固件 [小林固件]极米Z4X第三方官改，极致精简、去广告、ROOT，当贝桌面版，无法切换信号源。因为只有极米桌面才有信号源功能。联系卖家 QQ 群后，给了我新的固件，使用极米原版的 Launcher，且没 root。以后再也不买这种垃圾，还不如自己 root 后删系统应用呢。好像不 root 能用 ADB 也可以。

  1. 刷入上面提到的小林固件，jimi-z4-root-dangbei.zip
  2. 安装 3C All-in-One Toolbox-Pro-v2.2.2c_build_200111-Mod-armeabi-v7a.apk 和极米桌面
  3. 用 3C 删掉当贝的那堆垃圾，其它系统应用也可以删，可以只留系统设置和无线投屏
  4. 配置极米桌面，删掉不用的栏目
  5. 用 3C ，应用管理 - 极米桌面 - 打开 - Open external Directory，删掉 cache 文件夹里面的内容，同时去掉 cache 的写入权限，这样极米桌面就没有下载的图片了
  6. 不要用这个。用 3C ，应用管理 - 极米桌面 - 打开 - 打开数据目录，databases 中的 okgo_server.db 如果被去掉所有权限，就能做到桌面上什么都没有，信号源功能还可用。原本打算这样然后再装一个 TV Launcher，但尝试后发现，在 Top_TV_Launcher 中无法看到极米桌面（其它 Launcher 也看不到），而不启动极米桌面就无法使用信号源，╮（╯＿╰）╭ 。而且使用其它 Launcher 再启动极米桌面会有一段延迟。看来只能是有裁剪的极米桌面才能达到我想要的效果
