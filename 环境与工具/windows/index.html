搭建

  1. 安装 PE 环境到 U 盘，PE 环境尽量不更新，否则 DG 版本更新后之前的系统备份是否还能兼容

  2. PE 环境下加载 ISO 镜像，运行 source 下的 setup

  3. 激活

  4. 系统配置

    • 网络设置为私有网络，共享中心中开启发现和文件共享。右键文件夹授予访问权限，选择当前用户。两个 PC 是同样的用户密码，不知道不相同时能否实现共享
    • 手动休眠。电源选项中关闭睡眠和硬盘的超时，管理员 cmd powercfg.exe /hibernate off 关闭休眠。新建快捷方式 rundll32.exe powrprof.dll,SetSuspendState 0,1,0，并设置快捷键 ctrl-alt-s，拷入开始菜单。快捷键占用参考
    • 修改文档、音乐、视频、桌面等系统文件夹位置
    • 导入注册表相关设置，见 reg 文件夹
    • 上传账户头像，设置自动登录，Windows+R 运行 control userpasswords2
    • 安装 sans、serif、mono、Consolas-with-Yahei Nerd Font 字体和主题，见 必装/字体&主题 文件夹
    • 建立绿色软件的文件类型关联，将快捷方式添加到开始菜单

备份与还原

安装大型软件前一定备份

  1. DG 备份

    支持增量备份，占用空间小，备份容易管理

  2. Windows 自带备份

    占用空间大，但可靠，正常只保留一两个最新备份，作为 DG 还原失败的第二手段

    如果有启动项或服务在其它分区，那么其它分区也一起备份，这个太局限了。

    备份管理不方便，只能全删或者保留最近的一个

    注意还原时不选择从还原点还原，选择从备份驱动器还原

  3. dism++ 不要用，备份太慢了

  4. ghost 不能用

  5. Acronis True Image 听说速度和大小都不错，对于 PE 也有精简版，略麻烦没尝试过
